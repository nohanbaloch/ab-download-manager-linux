================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Source application not found at /opt/ABDownloadManager"
Solution: Build and place application at that location first

Problem: "Permission denied" when running build-deb.sh
Solution: chmod +x build-deb.sh && ./build-deb.sh

Problem: Application doesn't appear in menu
Solution: sudo update-desktop-database /usr/share/applications

Problem: Icon not showing
Solution: sudo gtk-update-icon-cache -f -t /usr/share/icons/hicolor

Problem: Dependency installation errors
Solution: sudo apt install -f

Problem: Need to reinstall
Solution: sudo dpkg -r abdownloadmanager && ./build-deb.sh && sudo dpkg -i abdownloadmanager_1.8.8_amd64.deb
